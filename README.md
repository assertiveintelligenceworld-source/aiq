# AIQ — Assertive Intelligence Quotient

AIQ is a public, open framework for measuring and cultivating **Assertive Intelligence**:
the ability to express truth clearly while maintaining dignity, stability, and mutual respect.

## Charter (AIQ憲章)
https://assertiveintelligenceworld-source.github.io/aiq.world/charter.md

## AIQ World

## What this repository is
This is the **collaboration workspace** for AIQ:
- protocols and reproducible analysis templates
- governance notes
- volunteer coordination (Issues / Pull Requests)

## Founder position
Initiated by **average**.
The initiator is an **observer** (non-executive).
No government, corporation, or individual may claim exclusive ownership or control of AIQ.

## How to help
- Open an Issue: ideas / protocols / translation / statistics / ethics / governance
- Submit a Pull Request if you can improve docs or tooling
